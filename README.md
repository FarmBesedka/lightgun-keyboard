# Световой пистолет для клона Duck Hunt на PixiJS

**Arduino Pro Micro + фоторезистор + кнопка**

---

## 🎯 Принцип работы

Пистолет эмулирует клавиатуру. При нажатии на курок отправляется клавиша `F23` (выстрел). При срабатывании фоторезистора (попадание в белую цель) — клавиша `F24` (подтверждение попадания).

---

## 🔌 Подключение

| Компонент       | Пин Arduino |
|----------------|-------------|
| Курок          | 2           |
| Фоторезистор   | 3           |
| GND            | GND         |

---

## 🎮 Используемые клавиши

- **F23** — выстрел
- **F24** — подтверждение попадания (фоторезистор)
